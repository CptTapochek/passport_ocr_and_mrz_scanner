# Passport scann data with ocr and validate it with nfc tag 

Proccess: 
  1) If you want to get data from the passport through nfc tag you have to decrypting this with [passport number, date of birth, date of expiry]
  2) When you open the app you have two options, to put data from passport[passport number, date of birth, date of expiry] manualy or through ocr scanner with camera.
  3) After that you have the data you can scann the nfc tag to validate that, and recive all data, including the photo
